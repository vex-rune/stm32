# STM32 术语缩写字典

> 数据来源：[基础篇](./基础篇/) 与 [进阶篇](./进阶篇/) 中出现的术语。同一缩写在不同上下文有多个含义时分别列出。

## 一、芯片与内核

| 缩写 | 全称 / 含义 |
| --- | --- |
| STM32 | ST 公司基于 ARM Cortex-M 内核的 32 位微控制器系列 |
| ARM | Advanced RISC Machines，处理器架构与设计公司 |
| Cortex-M3 | ARM 面向微控制器的 Cortex-M 系列内核之一 |
| MCU | Micro Controller Unit，微控制器 |

## 二、时钟与复位

| 缩写 | 全称 / 含义 |
| --- | --- |
| HSI | High Speed Internal（oscillator），高速内部时钟（8 MHz RC 振荡器） |
| HSE | High Speed External（oscillator / clock），高速外部时钟（通常外接 8 MHz 晶振） |
| LSI | Low Speed Internal，低速内部时钟（约 40 kHz，给看门狗 / RTC） |
| LSE | Low Speed External oscillator，低速外部时钟（32.768 kHz 晶振，给 RTC） |
| PLL | Phase Locked Loop，锁相环 / 倍频器 |
| SYSCLK | System Clock，系统时钟 |
| HCLK | AHB 总线时钟 |
| PCLK1 | APB1 总线时钟（低速外设） |
| PCLK2 | APB2 总线时钟（高速外设） |
| RCC | Reset and Clock Control，复位与时钟控制（控制器） |

## 三、中断与异常

| 缩写 | 全称 / 含义 |
| --- | --- |
| NVIC | Nested Vectored Interrupt Controller，嵌套向量中断控制器 |
| EXTI | External Interrupt / Event Controller，外部中断 / 事件控制器 |
| IRQ | Interrupt ReQuest，中断请求 |
| ISR | Interrupt Service Routine，中断服务程序 |
| AFIO | Alternate Function I/O，复用功能 I/O |

## 四、GPIO 与寄存器

| 缩写 | 全称 / 含义 |
| --- | --- |
| GPIO | General-Purpose Input / Output，通用输入输出口 |
| IDR | Input Data Register，输入数据寄存器 |
| ODR | Output Data Register，输出数据寄存器 |
| BSRR | Bit Set/Reset Register，位置位 / 复位寄存器 |
| BRR | Bit Reset Register，位复位寄存器 |
| CRL | Port Configuration Register Low，端口配置寄存器低 |
| CRH | Port Configuration Register High，端口配置寄存器高 |

## 五、总线与存储

| 缩写 | 全称 / 含义 |
| --- | --- |
| AHB | Advanced High-performance Bus，高速总线 |
| APB | Advanced Peripheral Bus，外设总线 |
| FSMC | Flexible Static Memory Controller，灵活的静态存储器控制器 |
| RAM | Random Access Memory，随机存取存储器 |
| ROM | Read Only Memory，只读存储器 |
| EPROM | Erasable Programmable ROM，可擦除可编程 ROM |
| PROM | Programmable ROM，可编程 ROM |
| SRAM | Static RAM，静态随机存取存储器 |
| PSRAM | Pseudo SRAM，假（伪）静态随机存取存储器 |
| NOR FLASH | 或非型闪存 |

## 六、通信外设

| 缩写 | 全称 / 含义 |
| --- | --- |
| USART | Universal Synchronous / Asynchronous Receiver / Transmitter，通用同步 / 异步收发器 |
| UART | Universal Asynchronous Receiver / Transmitter，通用异步收发器 |
| SPI | Serial Peripheral Interface，串行外设接口 |
| I2C | Inter-Integrated Circuit，集成电路总线 |
| SDA | I2C 数据线 |
| SCL | I2C 时钟线 |
| MOSI | Master Out Slave In，SPI 主机输出从机输入 |
| MISO | Master In Slave Out，SPI 主机输入从机输出 |
| SCK | Serial Clock，串行时钟 |
| CS / NSS | Chip Select / Negative Slave Select，片选 |
| CAN | Controller Area Network，控制器局域网 |

## 七、定时器与模拟

| DMA | Direct Memory Access，直接存储器访问 |

## 备注

- 本字典由脚本扫描基础篇 / 进阶篇后人工筛选整理，仅收录语义清晰的术语与缩写；代码标识符（函数名、寄存器位定义、引脚号等）已剔除。
- 部分缩写（如 SPI、I2C）在文档中既出现"接口协议"也出现"硬件外设模块"两种含义，本表取其主要含义。
- 含义行若截断（来自原文表格行），以"…"略去，详见对应章节源文件。
